# 🖥️ 투민시스템 홈페이지 관리 및 운영 매뉴얼 (초보자용)

이 문서는 웹 프로그래밍이나 서버 관리를 잘 모르는 초보자도 **투민시스템 홈페이지(twominsystem.com)**를 손쉽게 수정하고, 관리하며, 이메일 문의를 처리할 수 있도록 작성된 인수인계 지침서입니다.

---

## 📂 1. 홈페이지 구조와 작동 원리

투민시스템 홈페이지는 **정적 웹사이트(Static Website)**로 제작되었습니다. 데이터베이스 서버가 따로 없고, 모든 화면 정보와 제품 정보가 웹 문서(HTML) 안에 직접 기록되어 작동합니다.

* **호스팅 서비스 (서버)**: **GitHub Pages (깃허브 페이지)**
  * 비용이 완전히 무료이며, 트래픽 제한이 거의 없어 대량 접속 시에도 안정적입니다.
* **도메인**: `twominsystem.com` (폴더 내 `CNAME` 파일이 도메인 연결을 담당합니다.)
* **핵심 파일 및 폴더 구조**:
  * [index.html](file:///E:/업무/투민 홈페이지/홈페이지 자료/twoomin/index.html): 메인 페이지 (브랜드 소개, 제품 리스트, B2B 견적 문의 양식 탑재)
  * [product-detail.html](file:///E:/업무/투민 홈페이지/홈페이지 자료/twoomin/product-detail.html): 제품 상세페이지 (제품 코드 파라미터를 받아 정보를 동적으로 보여주는 공용 템플릿)
  * **이미지 파일들 (`.jpg`, `.png`)**: 웹사이트에 표시되는 제품 사진 및 인포그래픽 상세 이미지들 (폴더의 루트 경로에 직접 위치)
  * [publish.bat](file:///E:/업무/투민 홈페이지/홈페이지 자료/twoomin/publish.bat): 수정된 내용을 인터넷 서버에 반영해 주는 자동 배포 프로그램

## ✍️ 2. 홈페이지 내용 및 제품 정보 수정하기

가장 기본적인 글자 수정 및 이미지 수정 방법입니다. 파일을 수정할 때는 **Notepad++**나 **VS Code** 같은 무료 코드 편집기를 사용하시는 것을 권장합니다. (기본 메모장도 가능하지만 인코딩이 깨질 수 있으니 주의해 주세요.)

### ① 메인 페이지 글자 수정하기
1. [index.html](file:///E:/업무/투민 홈페이지/홈페이지 자료/twoomin/index.html) 파일을 편집기로 엽니다.
2. 바꾸고 싶은 글자(예: 주소, 전화번호 등)를 `Ctrl + F`로 검색하여 찾아 수정합니다.
3. 파일을 저장(`Ctrl + S`)합니다.

### ② 상세페이지 제품 데이터베이스(DB) 수정하기
상세페이지의 제품 스펙이나 상세 인포그래픽 이미지를 바꾸려면 [product-detail.html](file:///E:/업무/투민 홈페이지/홈페이지 자료/twoomin/product-detail.html) 파일 하단에 있는 **`productDb`** 자바스크립트 데이터를 수정해야 합니다.

> [!NOTE]
> `product-detail.html` 파일의 약 580번 라인부터 다음과 같은 형식의 **제품 데이터베이스**가 작성되어 있습니다.

```javascript
const productDb = {
    'TA300': {
        name: 'TA300 RGB',                // 제품 이름
        category: '케이스',                // 분류
        badge: 'FLAGSHIP PREMIUM',        // 제품 뱃지 태그
        subtext: '경계를 허물다. 무결점 프레임리스...', // 서브 설명글
        colors: ['black', 'white'],       // 제공 색상 (색상 버튼 생성 기준)
        img: {
            'black': 'TA300_RGB_Black_1000x1000.jpg', // 블랙 썸네일
            'white': 'TA300_RGB_White_1000x1000.jpg'  // 화이트 썸네일
        },
        detailImgs: {
            'black': 'TA300_RGB_Black_20260120.jpg', // 블랙 상세 인포그래픽
            'white': 'TA300_RGB_White_20260120.jpg'  // 화이트 상세 인포그래픽
        },
        quickSpecs: [
            { label: '규격', val: 'ATX 미들타워 (파노라마 뷰)' }, // 요약 스펙 리스트
            ...
        ],
        specs: {
            '제품 분류': 'PC 케이스 (ATX 미들타워)',               // 스펙 표 항목들
            '메인보드 지원': 'Standard ATX, Micro-ATX, Mini-ITX',
            ...
        }
    },
    ...
}
```

* **상세 인포그래픽 이미지 변경 시**: `detailImgs` 항목에 교체할 새로운 이미지 파일명(예: `TA100_black_new.jpg`)을 따옴표 안에 적어주면 작동합니다.

## 🖼️ 3. 이미지 추가 시 주의사항 및 최적화

홈페이지 조작 미숙으로 사이트 속도가 매우 느려지는 가장 큰 원인은 **이미지 용량** 때문입니다.

> [!IMPORTANT]
> **이미지 업로드 가이드라인**
> * **포맷 규격**: 상세 인포그래픽 이미지는 원본 포맷이 `.png`라 하더라도 가급적 **`.jpg` 포맷으로 변환**하여 사용하세요.
> * **용량 최적화**: 10MB가 넘는 고해상도 이미지는 로딩 속도를 매우 떨어뜨립니다. 무료 온라인 이미지 압축 사이트(예: [iloveimg.com](https://www.iloveimg.com/ko/compress-image) 등)를 이용하여 장당 **용량을 3MB 이하(가급적 1MB 대)로 줄여서** 업로드해 주세요.
> * **파일명 매칭**: 대소문자를 구분하므로 코드에 적은 파일명과 실제 이미지 파일명이 완벽히 일치해야 엑스박스(`X`) 현상이 생기지 않습니다. (예: `ta100.jpg` 와 `TA100.JPG`는 서로 다름)

## 📧 4. B2B 이메일 문의 접수 시스템

고객이 홈페이지 하단 견적 양식에 작성하면 메일로 수신되는 시스템입니다. 외부 메일 발송 API 서비스인 **FormSubmit.co**를 사용하고 있습니다.

* **문의 이메일 수신 방식**: 
  * 문의가 들어오면 `psh@hammertech.co.kr` 메일 주소로 전송되며, 서브 담당자인 `rlaqnwnd1@hanmail.net`으로도 개별 동시 수신(CC 기능 대체)됩니다.
* **간편 답장 기능**:
  * 고객이 본인의 이메일을 적어서 제출하면, 수신된 이메일 헤더의 `Reply-To` (회신 주소)에 고객의 이메일 주소가 자동으로 들어갑니다.
  * 따라서 관리자 메일함에서 수신된 문의 메일을 열고 **[답장]** 버튼을 누르기만 하면 문의한 고객에게 바로 회신 메일을 작성하실 수 있습니다.

## 🚀 5. 수정한 내용을 서버에 배포하기 (가장 중요)

로컬 컴퓨터에서 코드를 수정했거나 이미지를 폴더 안에 넣었다면, 이를 서버에 적용시켜 세상에 노출해야 합니다. 복잡한 명령어를 칠 필요 없이 자동화 도구를 실행하면 됩니다.

1. 업데이트할 이미지 파일을 윈도우 폴더(`E:\업무\투민 홈페이지\홈페이지 자료\twoomin`) 안에 복사해 넣습니다.
2. 폴더 내에 있는 **`publish.bat`** 파일을 마우스 더블클릭으로 실행합니다.
3. 검은색 명령 창이 실행되며 파일 체크를 수행한 후, 아래와 같은 메시지가 뜹니다.
   > **"수정 내용을 간단히 입력하세요 (예: 이미지 추가) [기본값: 수정함]:"**
4. 수정 사항을 짧게 입력하고 **Enter(엔터)**를 칩니다. (예: `TA100 상세페이지 수정`)
5. 검은 창에 진행 상황이 출력되며 전송이 성공하면 **"업데이트 완료! 약 1~2분 뒤 반영됩니다."**라는 문구가 뜹니다.
6. 아무 키나 눌러 창을 닫습니다.

> [!WARNING]
> **캐시(Cache)로 인한 미반영 문제 조치법**
> 배포 후 1~2분이 지나도 스마트폰이나 PC에서 변경 사항이 보이지 않는다면 브라우저 캐시 때문입니다.
> * **PC 크롬/엣지 브라우저**: 키보드의 **`Ctrl + Shift + R`** (강력 새로고침)을 누르시면 최신 상태로 갱신됩니다.
> * **스마트폰 브라우저**: 브라우저 설정에서 "인터넷 사용 기록 삭제" -> "캐시된 이미지 및 파일 삭제"를 한 후 새로고침해야 새 화면이 보입니다.

## 🛠️ 6. 긴급 상황 대처법 (FAQ)

### Q. 이미지를 올렸는데 상세페이지 하단에 엑스박스(X)로 깨져서 나와요.
* **원인 1**: 이미지 파일명과 소스코드에 등록한 파일명의 글자가 완전히 동일한지 확인하세요. (예: `TA100_Black.jpg` vs `TA100_black.jpg` 대소문자 확인)
* **원인 2**: `publish.bat`를 돌릴 때 새로 추가한 이미지 파일이 해당 폴더에 제대로 안 들어있었을 수 있습니다. 파일을 다시 확인하고 배포를 다시 수행하세요.

### Q. 이메일 문의가 갑자기 수신되지 않아요.
* **원인**: 홈페이지가 처음으로 다른 이메일 주소로 전송을 요청할 경우, 스팸 차단을 막기 위해 FormSubmit 서비스로부터 **최초 1회 이메일 인증 메일**이 발송됩니다. 
* **해결**: 수신자 메일함(`psh@hammertech.co.kr`)의 스팸함 또는 받은메일함을 확인하여 FormSubmit에서 보낸 인증 메일의 **[Active Form]** 링크 버튼을 클릭해 활성화해 주시면 정상 접수되기 시작합니다.
